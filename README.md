## 📖 About

This repository contains my personal developer portfolio built with **React**, **Vite**, and **TypeScript**.

The portfolio is designed as a modern, single-page experience featuring smooth animations, clean typography, responsive layouts, and scroll-based storytelling. It highlights my projects, technical skills, internship experience, education, and contact information.

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/laptopah16-art/Portfolio-ANAND.git
```

Navigate to the project

```bash
cd Portfolio-ANAND
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview the production build

```bash
npm run preview
```
